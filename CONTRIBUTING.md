# Contributing

This is a personal reference / archive repository. It captures a working pattern from a real CDC pipeline and is not actively maintained.

If you spot a bug, have a question, or want to suggest a change:

1. Open a GitHub issue describing what you found or what you'd like to discuss.
2. For changes, fork the repo and open a pull request against `main`. Keep the diff focused on a single concern.
3. Note that the upstream Flink and CDK versions used here are dated; pull requests that modernize the build are welcome but require corresponding test updates.

There is no CLA. By submitting a contribution you agree it is licensed under the MIT-0 license in `LICENSE`.
